# 类之间
不同的形状的连接器表示类之间的不同关系：
空心的三角形箭头，指向父类 -> 表示继承关系
单调的直线  -> 表示类之间有关联，但是仅此而已了，不会影响。
一个空心菱形，指向的为整体 -> 表示整体与个体的关系，比如一头狼和一只狼群，这就是一种聚合关系。 但是狼也可以脱离狼群存在。
一个实心菱形，指向的为整体 -> 这是组合关系：比如一个订单上有多个订单项，就是整体和个体的关系。 但是订单项不能脱离订单，这就是组合关系。
